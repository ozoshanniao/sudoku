## 🌟 核心功能与特性 (Features & Highlights)

*   **经典数独游戏**: 支持多种难度选择，包括简单、中等、困难和专家模式，满足不同阶段玩家的需求。
*   **每日挑战 (Daily Challenge)**: 每天解锁新的特定数独谜题，挑战你的连续打卡记录。
*   **智能进度保存**: 自动保存游戏进度，随时随地退出，下次打开无缝恢复你未完成的棋局。
*   **玩家等级与经验系统 (XP & Profile)**: 通过完成游戏赚取 XP，提升你的玩家等级并解锁不同阶段的称号与成就。
*   **详细数据统计 (Stats)**: 记录你的总胜场、最佳时间、完美通关次数，直观呈现你的成长轨迹。
*   **流畅的交互与极简视效**: 充满细节的微动画、清晰纯净的界面设计，为你带来丝滑且舒适的视觉体验。
*   **个性化自定义设置**: 支持游戏音效开关等基础设置，让游戏体验更符合你的个人习惯。
